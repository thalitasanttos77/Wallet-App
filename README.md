# Wallet App 💰

O **Wallet App** é uma aplicação Android nativa desenvolvida para auxiliar utilizadores na gestão das suas finanças pessoais de forma simples e intuitiva. O projeto foca-se em fornecer uma interface limpa para o registo e visualização de transações financeiras.

## 🚀 Funcionalidades

* **Gestão de Carteira:** Visualize o saldo atual de forma rápida.
* **Registo de Transações:** Adicione entradas e saídas financeiras.
* **Histórico Detalhado:** Acompanhe todas as movimentações realizadas.
* **Interface Moderna:** Design focado na experiência do utilizador (UX/UI).

## 🛠️ Tecnologias Utilizadas

* **Linguagem:** [Kotlin](https://kotlinlang.org/) (100%)
* **Build System:** Gradle (Kotlin DSL)
* **Arquitetura:** MVVM (Model-View-ViewModel) - *Sugerido com base em padrões modernos de Android*
* **UI:** XML ou Jetpack Compose

## 📦 Como instalar e executar

Para rodar este projeto localmente, precisará do **Android Studio** instalado.

1. **Clone o repositório:**
```bash
git clone https://github.com/thalitasanttos77/Wallet-App.git

```


2. **Abra o projeto:**
* Inicie o Android Studio.
* Selecione `Open` e navegue até a pasta onde clonou o repositório.


3. **Sincronize o Gradle:**
* Aguarde que o Android Studio descarregue todas as dependências necessárias.


4. **Execute a App:**
* Ligue um dispositivo físico via USB ou utilize um Emulador.
* Clique no botão `Run` (ícone de play verde).



## 📂 Estrutura do Projeto

O projeto segue a estrutura padrão do Android:

* `/app`: Contém o código fonte e os recursos da aplicação.
* `/gradle`: Ficheiros de configuração do motor de build.
* `build.gradle.kts`: Configurações de dependências e do projeto.
